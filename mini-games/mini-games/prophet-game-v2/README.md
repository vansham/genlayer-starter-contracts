# GenLayer Prophet v2

Multiplayer Optimistic Consensus Game for GenLayer Community.

GenLayer Prophet v2 is a fun educational mini-game where players act as validators in a consensus system.  
Players submit predictions, majority vote becomes the truth, and XP is distributed based on accuracy and speed.

## How to Run

1. Download the repository
2. Open `index.html` in any browser
3. Create a room and invite friends using the room code
4. Play and experience consensus in action

## Features

- Multiplayer rooms
- 6 rounds per game
- Speed-based XP rewards
- Final leaderboard
- Weekly question sets
- Fully client-side, no backend required
